How does it work?

  1. The code maker select a 4 colors code (the colors can repeat themselves)
  2. You have 10 guesses to find the code: the colors and their locations
  3. First guess: set up the code on the first row and click 'check'
    to see if you were right
  4. Feedback: Next to each "guess row" is a small square with four tiny dots.
    • Each orange dot means that one of the guessed colors is correct, but is in the wrong hole.
    • Each green dot means that one of the guessed colors is correct, and is in the right hole.
    • The order of the gray and green dots does not matter.
  5. Use the feedback from your previous try to guess again
  6. Continue until the code is guessed or there are no more guesses left.

Good Luck!
